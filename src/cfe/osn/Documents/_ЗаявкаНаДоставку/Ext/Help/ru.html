Заявка на доставку

Заявка на доставку - это заявка  службе отгрузки сфомироваит складские ордера  на поставку клиенту товаров или оказание услуг в установленные сроки. Заявка на доставку может быть оформлен на основании заказ(ов) клиента.

Оформление заявки

В Заявка на доставку  обязательно должны быть указаны:Клиент;

  • Способ достави
  • Дата доставки (отгрузки).

Способ доставки

Возможность выбора в документе того или иного способа доставки определяется как будет доставлен товар клиенту :

  • Самовывоз (с нашего склада);
  • Наша транспортная служба до клиента.


Способ доставки и группа реквизитов доставки устанавливаются для всез выбранных заказов клиента.

Группа доставки

При выборе способа доставки "Наша транспортная служба до клиента

" в группе доставки будут доступна для заполнения следющие поля:
  • Зона доставки;
  • Адрес доставки:
    • заполняется из данных клиента "Синяя стрелка рядом с полем адреса";
    • выбором из стандарного классификатора адресов;
  • Дополнительная информация по доставке:
    • заполняется из данных клиента "Синяя стрелка рядом с полем Доп.информации", при заполнении поля "Контактное лицо" страницы "Дополнительно" будут внесены данные по контактному лицу;
    • вручную прописывается необходимая информация для Отдела доставки;
  • Если необходимо указать дополнительну выжнуж информация для Отдела доставки, установить отметку "Обратить внимание транспортной службе" и заполнить поле "Особые условия перевозки";
  • Указать Отделу Доставки время доставки товара коиенту С и ПО.
